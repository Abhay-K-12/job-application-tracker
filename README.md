# Job Application Tracker 📋

A simple Python CLI tool to track job applications with status, date, and links.

## 🚀 Features
- Add job applications (company, role, status)
- Automatically stores date of application
- Save job link for reference
- View all applied jobs
- Helps organize job search process

## 🛠️ Tech Used
- Python
- CSV file handling

## ▶️ How to Run

1. Clone the repository:
git clone https://github.com/YOUR_USERNAME/job-application-tracker.git

2. Navigate to folder:
cd job-application-tracker

3. Run the program:
python job_tracker.py

## 💡 Why I Built This

While applying for jobs, I found it difficult to track where I had applied and the status of each application.

This tool helps organize job applications efficiently and reduces confusion during the job search process.
